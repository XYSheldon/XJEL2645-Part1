107 - C++ Standard Library

Learning Outcomes

By the end of this session, students should:

  • Have an appreciatation of the C++ standard library.
  • Be comfortable with using some of the most common C++ standard library classes.
  • Be able to indendently research online how to use new standard library classes through finding and reading API's and example code.
